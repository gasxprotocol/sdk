# @gasx/contracts

ABIs + canonical addresses for the **GasX** modular ERC-4337 paymaster contracts
(`gasxprotocol/contracts`): `GasXPolicyManager`, `GasXPaymasterBase`, `GasXWhitelistPaymaster`,
`GasXERC20FeePaymaster`, and the `IGasX*` interfaces — plus the canonical EntryPoint v0.9 address.

```bash
npm i @gasx/contracts
```

```ts
import { GasXPolicyManagerAbi, ENTRYPOINT_V09 } from "@gasx/contracts";
```
